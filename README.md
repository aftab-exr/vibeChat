# VibeChat 💬📸🎤📞

A full-stack real-time messaging and calling application built using **Flutter + Node.js + Socket.io + WebRTC**.

---

## 🚀 Features

* 💬 Real-time chat (Socket.io)
* 📸 Image sharing
* 🎤 Voice messages
* 📞 Voice calling (WebRTC)
* 🎥 Video calling (WebRTC)
* 🔐 JWT Authentication
* ⚡ Lightweight SQLite database

---

## 🧠 Tech Stack

| Layer        | Technology        |
| ------------ | ----------------- |
| Frontend     | Flutter           |
| Backend      | Node.js (Express) |
| Database     | SQLite            |
| Real-time    | Socket.io         |
| Calls        | WebRTC            |
| Media Upload | Multer            |

---

## 📂 Project Structure

```
vibeChat/
 ├── backend/        # Node.js API + Socket server
 ├── flutter_app/    # Flutter mobile app
 ├── docs/           # Screenshots & demo
```

---

## ⚙️ Setup Instructions

### 🔧 Backend

```bash
cd backend
npm install
node server.js
```

---

### 📱 Flutter App

```bash
cd flutter_app
flutter pub get
flutter run
```

---

## ⚠️ Configuration

Replace local IP in Flutter & backend:

```
http://192.168.x.x:4000
```

👉 Use your machine's IP address
